# 高端利率计算器 (Interest Calculator)

本项目是一个使用纯前端技术（HTML5, Vanilla CSS, Vanilla JavaScript）打造的高质感、现代化的静态利率计算器网页。它支持单利和复利的计算，并提供直观的可视化结果展示和多级本地化支持。

## 🎯 目标与核心功能

- **输入参数支持**：首付/本金、年化利率（最高可达 100%）、投资期限（年）、计息方式（单利 / 复利）。
- **复利频率选择**：如果选择复利，可进一步选择复利计算频率（每年、每半年、每季度、每月、每日）。
- **即时计算反馈**：用户修改任何参数时，结果会实时更新，包括图表的动画，无需点击“计算”按钮。
- **现代化设计语言**：
    - 采用高端暗色模式 (Sleek Dark Mode) 结合 玻璃拟物化 (Glassmorphism) 的深色卡片设计风格。
    - 使用 Google Fonts 现代无衬线字体 (Outfit)。
    - 平滑的控制面板滑动条，提供优良的用户体验。
- **动态数据可视化**：通过引入 [Chart.js](https://www.chartjs.org/) 提供高质感的堆叠柱状图，直观展示本金与利息按年的增长与比例。
- **完全国际化 (i18n)**：包含原生的多语言切换功能支持（中文、English、日本語、한국어、Español），实时根据选择格式化货币符号及刻度逻辑。

## 📁 项目结构

本项目为纯粹的前端静态项目，开箱即用，所有源代码放置在单一目录下：

- `index.html`：核心 DOM 骨架视图，包含了所有输入控件的布局和语言选择器，以及引入相关的外部 CDN 依赖。
- `styles.css`：全部视觉样式表，采用 CSS 原生属性，全局变量管理颜色/玻璃质感/间距。
- `script.js`：涵盖事件绑定、翻译字典（五国语言）、单/复利的数学渲染逻辑，并调用 Chart.js 进行按年投资视图渲染。

## 🚀 部署与体验方法

本应用为纯客户端静态网页，没有后端及网络服务绑定（仅需读取 CDN 以加载图表及字体资源）。
**体验步骤**：
直接下载本项目并在浏览器（推荐 Chrome, Firefox, Edge, Safari）中双击打开 `index.html` 文件即可立即使用。不需要 `npm install` 亦无需开启额外的开发服务器。

## 🧠 计算公式说明

1. **单利 (Simple Interest):**
   > $A = P(1 + rt)$
   
2. **复利 (Compound Interest):**
   > $A = P(1 + r/n)^{nt}$

   *(其中 A = 最终金额, P = 本金, r = 年利率, n = 每年复利次数, t = 投资年限)*
